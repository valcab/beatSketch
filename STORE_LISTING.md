# BeatSketch Store Copy

This file contains draft presentation text for the Chrome Web Store in English and French.

## English

### Name

`BeatSketch`

### Short Description

Sketch drum grooves fast with a compact Chrome drum machine built for guitarists and bassists.

### Promotional Summary

BeatSketch helps guitarists and bassists build quick drum ideas in a compact Chrome popup, using genre-based templates, dual A/B patterns, tap tempo, and a simple step sequencer.

### Why This Extension Exists

When writing riffs, practicing, or testing ideas, opening a full DAW can feel too heavy.  
BeatSketch was created as a lightweight rhythm sketchpad: open the popup, pick a groove, tweak the pattern, and play immediately.

### What It Does

- Program beats in a compact step sequencer
- Switch between two patterns: `A` and `B`
- Queue pattern changes at the end of the current bar
- Start from genre-based templates across multiple styles
- Adjust BPM quickly with slider, numeric input, or tap tempo
- Edit per-step velocity
- Paint multiple steps by clicking and dragging
- Use built-in sample playback with fallback synthesis if samples are missing
- Save and reload beats locally
- Control master volume and master reverb from the popup

### Key Benefits

- Faster than opening a full music production app
- Useful for riff writing, practice, and arrangement sketching
- Compact enough to stay inside a browser workflow
- Built around simple drummer-like pattern switching
- Works even when some drum samples are unavailable

### Detailed Description

BeatSketch is a Chrome extension drum machine designed for guitarists, bassists, and songwriters who want a fast way to sketch rhythm ideas without leaving the browser.

The extension opens as a compact popup and gives you a practical step sequencer with genre templates, pattern A/B switching, tap tempo, and quick transport controls. It is especially useful for practice sessions, riff exploration, backing groove ideas, or rough songwriting drafts when launching a full DAW would interrupt the flow.

BeatSketch includes a Web Audio-based playback engine, synthetic convolution reverb, per-step sequencing, and a lightweight workflow centered on speed. If some drum WAV files are missing, the extension still remains playable thanks to built-in fallback drum synthesis.

Main features:

- Popup-based step sequencer
- Dual pattern workflow with A/B switching
- Queued pattern changes applied at the end of the bar
- Genre template library
- Tap tempo and BPM control
- Per-step activation and velocity editing
- Click-drag pattern painting
- Local save/load
- Web Audio playback
- Master volume and reverb controls

BeatSketch is meant to stay focused, immediate, and musical: open it, sketch a groove, and keep playing.

### Privacy-Oriented Notes

- No account required
- No cloud sync required
- Data is stored locally in the browser
- No external dashboard or online service is required for core use

### Description Détaillée

BeatSketch est une extension Chrome de boîte à rythme conçue pour les guitaristes, bassistes et compositeurs qui veulent esquisser rapidement des idées rythmiques sans quitter le navigateur.

L’extension s’ouvre sous forme de popup compacte et propose un séquenceur pratique avec templates de genres, changement de pattern A/B, tap tempo et contrôles de transport rapides. Elle est particulièrement utile pour le travail de riffs, la pratique, les idées de backing groove ou les brouillons de composition quand ouvrir un DAW complet serait trop lourd.

BeatSketch intègre un moteur audio basé sur la Web Audio API, une reverb par convolution synthétique, l’édition pas à pas et un workflow pensé pour la rapidité. Si certains fichiers WAV sont absents, l’extension reste jouable grâce à un mode de synthèse de batterie de secours.

Fonctionnalités principales :

- Séquenceur pas à pas dans la popup
- Workflow dual pattern A/B
- Changement de pattern appliqué à la fin de la mesure
- Bibliothèque de templates par genre
- Tap tempo et contrôle du BPM
- Activation des pas et édition de vélocité
- Peinture des patterns en cliquant-glissant
- Sauvegarde et chargement local
- Lecture via Web Audio
- Contrôle global du volume et de la reverb

BeatSketch vise une expérience simple, rapide et musicale : ouvrir, créer un groove, continuer à jouer.

- Les données sont stockées localement dans le navigateur
- Aucun service externe n’est nécessaire pour l’usage principal
